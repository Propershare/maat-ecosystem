# MAAT Memory

Memory is not just vector search. Real memory has classes.

## Memory Classes

### Episodic (`episodic/`)
What happened. Conversations, events, interactions.
Time-bound. Decays. Can be consolidated into semantic.

### Semantic (`semantic/`)
What is known. Facts, knowledge, domain info.
Stable. Comes from consolidation or explicit storage.

### Constitutional (`constitutional/`)
Who I am. Values, identity, core beliefs.
**Non-reversible.** The only memory class that cannot be deleted.

### Task (`task/`)
What needs to be done. TODOs, decisions, plans.
Active. Gets completed or expired.

### Consolidation (`consolidation/`)
The process of turning episodic into semantic.
Happens periodically. Always reversible.
